Register Admin HR

URL

Method

POST

Request Body - Raw Json

   {
        "first_name" : string,
        "last_name" : string,
        "username" : string,
        "password" : string,
        "email" : string
   }

Response

    {
        "code" : 200
        "id": int,
        "message": "Admin HR registered successfully",
        "token": string
    }
